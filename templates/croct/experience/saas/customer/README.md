# Introduction

This template creates an experience to personalize your e-commerce website for users who are customers.

By segmenting customers from visitors, you can focus on nurturing loyalty and maintaining a solid relationship by providing exclusive benefits, rewards, and purchase history-based product recommendations.

## What's included

This experience includes:

- **Components:** CTA, announcement bar, and hero section
- **Slots:** announcement bar and home hero section
- **Audience:** `user's activities include "login" and user's plan is not "free"`

To create other audiences, you can use our [Contextual Query Language (CQL)](https://docs.croct.com/reference/cql/introduction). This English-based language combines the simplicity of natural language with an intuitive syntax, making it easy for both technical and non-technical users. You can, for instance, [enrich the user's profile](https://docs.croct.com/reference/sdk/javascript/data-collection) with first-party data and use [custom attributes](https://docs.croct.com/reference/cql/data-types/user). 

## Usage

To create a new experience using this template, run:

```js-pm
croct@latest use croct://experience/saas/customer
```
