# Introduction

Looking to personalize your site for **target accounts** using **6sense data**?

This integration connects [6sense](https://6sense.com) to Croct, letting you use account intent, firmographic, and 
buying-stage signals to power personalized experiences so marketers can tailor content to the companies that matter, 
without touching code.

> [!TIP Integration]
> Learn more about the Croct + 6sense integration by exploring the [integration page](https://croct.com/integrations/6sense).

Follow [this tutorial](https://docs.croct.com/immersion/integrations/firmographics/6sense/integration) to activate
6sense data in Croct and [this playbook](https://docs.croct.com/immersion/integrations/firmographics/6sense/playbook)
for ideas on how to put it to work.
