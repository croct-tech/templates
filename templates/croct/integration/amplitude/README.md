# Introduction

Want your product analytics to reflect what each visitor actually experienced?

This integration connects [Amplitude](https://amplitude.com?utm_source=croct) with Croct, forwarding the audiences,
experiences, experiments, and variants from every personalized or AB-tested session straight into your events.

It's the fastest way to enrich your product analytics with real personalization data, without changing how you already
track.

* Break funnels, retention, and cohorts down by audience, experience, experiment, and variant
* Measure the impact of every personalized experience and AB-test variant
* Keep your existing events, charts, and dashboards untouched
* No extra tracking code to write or maintain

## Usage

Run the following command in an empty directory to create a new project using this template:

```croct-cmd
croct use croct://integration/amplitude
```
