# Introduction

Need to deliver **account-based experiences** powered by **Demandbase data**?

This integration links [Demandbase](https://demandbase.com) with Croct, surfacing account identification, intent, and
engagement signals so your team can shape personalized experiences for high-value accounts straight from the marketing
dashboard.

> [!TIP Integration]
> Learn more about the Croct + Demandbase integration by exploring the [integration page](https://croct.com/integrations/demandbase).

Follow [this tutorial](https://docs.croct.com/immersion/integrations/firmographics/demandbase/integration) to activate
demandbase data in Croct and [this playbook](https://docs.croct.com/immersion/integrations/firmographics/demandbase/playbook)
for ideas on how to put it to work.
