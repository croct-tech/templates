# Introduction

Want your analytics to reflect what each visitor actually saw?

This integration connects [Google Analytics](https://marketingplatform.google.com/about/analytics?utm_source=croct)
with Croct, forwarding the audiences, experiences, experiments, and variants from every personalized or AB-tested
session straight into your reports.

It's the fastest way to enrich your growth analysis with real personalization data, without changing how you already
track.

* Break down reports by audience, experience, experiment, and variant
* Measure the impact of every personalized experience and AB-test variant
* Keep your existing events, goals, and dashboards untouched
* No extra tracking code to write or maintain

## Usage

Run the following command in an empty directory to create a new project using this template:

```croct-cmd
croct use croct://integration/google-analytics
```
