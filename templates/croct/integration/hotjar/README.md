# Introduction

Want to filter your session recordings by what each visitor actually saw?

This integration connects [Hotjar](https://www.hotjar.com?utm_source=croct) with Croct, forwarding the audiences,
experiences, experiments, and variants from every personalized or AB-tested session straight into Hotjar.

It's the fastest way to run qualitative analysis on your personalized experiences, without changing how you already
record.

* Filter recordings and heatmaps by audience, experience, experiment, and variant
* Watch how real users behave inside each personalized experience or AB-test variant
* Keep your existing recordings and heatmaps untouched
* No extra tracking code to write or maintain

## Usage

Run the following command in an empty directory to create a new project using this template:

```croct-cmd
croct use croct://integration/hotjar
```
