# Introduction

Want to turn your **Magento store activity** into **personalized experiences** with no extra development work?

This integration connects your [Magento](https://business.adobe.com/products/magento/magento-commerce) store to Croct,
automatically tracking store events such as product views, cart changes, checkouts, and orders, and enriching every user
profile with Magento customer data so marketers can build audiences, run AB tests, and personalize the storefront.

Follow [this tutorial](https://docs.croct.com/immersion/tutorials/magento-integration) to bring your Magento store data
into Croct and get started.
