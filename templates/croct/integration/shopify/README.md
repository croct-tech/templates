# Introduction

Want to turn your **Shopify store activity** into **personalized experiences** with no extra development work?

This integration connects your [Shopify](https://www.shopify.com) store to Croct, automatically tracking store events
such as product views, cart changes, checkouts, and orders, and enriching every user profile with Shopify customer data
so marketers can build audiences, run AB tests, and personalize the storefront.

Follow [this tutorial](https://docs.croct.com/immersion/tutorials/shopify-integration) to bring your Shopify store data into
Croct and get started.
