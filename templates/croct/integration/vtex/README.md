# Introduction

Want to turn your **VTEX store activity** into **personalized experiences** with no extra development work?

This integration connects your [VTEX](https://vtex.com) store to Croct, automatically tracking store events such as
product views, cart changes, checkouts, and orders, and enriching every user profile with VTEX customer data so
marketers can build audiences, run AB tests, and personalize the storefront.

It works with both VTEX storefront stacks, [Store Framework](https://developers.vtex.com/docs/guides/vtex-io-documentation-what-is-vtex-io)
and [FastStore](https://developers.vtex.com/docs/guides/faststore), and requires only a single snippet added to Google
Tag Manager.

Follow [this tutorial](https://docs.croct.com/immersion/tutorials/vtex-integration) to bring your VTEX store data into
Croct and get started.
