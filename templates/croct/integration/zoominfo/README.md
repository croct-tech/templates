# Introduction

Want to personalize your site based on **B2B intelligence** from **ZoomInfo**?

This integration brings [ZoomInfo](https://zoominfo.com) data into Croct, giving marketers access to company
profiles, intent signals, and technographic insights to craft tailored experiences for every visitor — all without
writing a single line of code.

Follow [this tutorial](https://docs.croct.com/immersion/integrations/firmographics/zoominfo) to activate ZoomInfo data
in Croct and [this playbook](https://docs.croct.com/immersion/playbooks/zoominfo) for ideas on how to put it to work.
